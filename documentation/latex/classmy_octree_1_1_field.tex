\subsection{Detailed Description}
Template class for any scalar field variable in the domain. 

Usage\+: \hyperlink{classmy_octree_1_1_field}{Field} object(nx,ny,nz); Parameters are the number of cells along x, y and z (including the padding) which usually remains same for all the blocks in the mesh. Padding represents extra layer of cells at all the sides of the block, which acts as ghost cells at the block boundaries or buffer cells at the processor boundaries. New objects of this class are defined in the constructors of \hyperlink{classmy_octree_1_1_block}{Block} class and the pointers to these objects are members of the \hyperlink{classmy_octree_1_1_block}{Block} class.
